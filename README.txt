Чтобы опробовать функционал приложения, нужно:

1) Поднять контейнер с бд. Команда: docker-compose up -d
2) Запустить приложение спринг-бут
3) Перейти в сваггер: http://localhost:8080/swagger-ui/index.html
4) Проверить функционал по созданию пользователя - эндпоинт /api/users/signup
5) Чтобы проверить остальной функционал, нужно зайти в систему под администратором.
